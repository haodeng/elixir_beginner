# Keyword lists
In Elixir, when we have a list of tuples and the first item of the tuple (i.e. the key) is an atom, we call it a keyword list

    iex> list = [{:a, 1}, {:b, 2}]
    [a: 1, b: 2]
    # Elixir supports a special syntax for defining such lists: [key: value]
    iex> list == [a: 1, b: 2]
    true
